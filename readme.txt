== Description ==

TPC! vCard allows WordPress administrators to import vCards and create new users from that information.

== Installation ==
1. Upload all files to the `/wp-content/plugins/tpc-vcard/` directory.
2. Activate the plugin via the Plugins menu within the WordPress administration panels.
3. Click on Import under vCard to get started.
